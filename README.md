# 4v4WarsWatcher

**Teamfight Tactics（TFT）の 4v4 対戦を観戦するためのオーバーレイアプリです。**
2チーム（各4人）の **合計HP**・**生存人数**・**各プレイヤーのチーム識別マーカー** を、
TFTのプレイヤーリスト上にリアルタイム表示します。

`Overwolf アプリ（UI・オーバーレイ）` ＋ `Python OCRワーカー（画面読み取り）` の2層構成です。

---

## 主な機能

- **チームごとの合計HP** … 各プレイヤーのHPをOCRで読み取り、チーム単位で合算
- **生存人数** … Overwolf GEP（ライブクライアントデータの `isDead`）から `生存/全体` を表示
- **チーム識別マーカー** … プレイヤーリストの各キャラ肖像に、チーム色のバッジ／アップロードした**チームアイコン**を重ねて表示
- **死亡者はHP=0扱い** … 脱落後に古いHPが加算され続けないよう補正
- **ホーム画面でチーム設定** … チーム名・メンバー（RiotID）・チームカラー・チームアイコンを設定
- **OCRリカバリ機能** … OCRが名前を読めない時のための手動補正
  - プレイヤー対応（読み取り画像を見て手動で名前を割り当て）
  - 行ごと／未読の一括 **再OCR**
  - **位置から登録**（上から数えた行のキャプチャを手動でテンプレート登録）

## 仕組み（アーキテクチャ）

```
              ┌──────────────── Overwolf アプリ ────────────────┐
TFT画面 ──▶  │ in_game(オーバーレイ): HP/生存/マーカーを描画        │
              │ home: チーム設定・ワーカー設定・OCRリカバリ          │
              │ background: ゲーム検知・ワーカー起動(C#プラグイン)    │
              │ GEP(live_client_data): 生存(isDead)・正式名          │
              └───────▲───────────────────────────┬───────────────┘
                      │ HTTP(127.0.0.1:17653)      │ 起動
                      │ /stats(HP・位置) /rows /assign など
              ┌───────┴───────────────────────────▼───────────────┐
              │ Python OCRワーカー (Windows)                        │
              │ WGCでTFTウィンドウをキャプチャ → 名前=EasyOCR /      │
              │ HP=Tesseract。NCCテンプレ照合(名前マスク＋肖像)で    │
              │ 並び替えに追従。結果をHTTPで配信                     │
              └────────────────────────────────────────────────────┘
```

- **チーム分け・色・合計・生存判定はOverwolf側**が担当
- **ワーカーは「誰が・画面のどのY位置で・何HPか」だけ**を返す純粋なOCRエンジン
- 名前の追跡は **NCC（正規化相互相関）テンプレート照合**：名前の白文字マスク＋タクティシャン肖像を併用し、背景差や短い名前にも強い

## 動作環境

- **Windows**（WGCキャプチャのため。WSL内では不可）
- **Overwolf**（開発者モードで Load unpacked して使用）
- **Python 3.12** ＋ 依存パッケージ（`worker/requirements.txt`）
- **Tesseract-OCR**（HP数字の読み取り。日本語データ `jpn` も推奨）
- 解像度 **1920×1080** 前提（マーカー座標が固定）

---

## セットアップ

### 1. OCRワーカー（Python）

```bash
cd worker
pip install -r requirements.txt
```

- Tesseract をインストール（[UB-Mannheim ビルド](https://github.com/UB-Mannheim/tesseract/wiki)）。
  PATHに無い場合は環境変数 `TESSERACT_CMD` でフルパス指定（アプリの設定からも指定可）。

### 2. Overwolfアプリ

1. Overwolfクライアントに**ログイン**し、**開発者モード**を有効化
2. 開発者メニューの **「Load unpacked extension」** で、このリポジトリの **`manifest.json` があるフォルダ**を選択
   - ※ ログインしていないと「Unauthorized App」になります

---

## 使い方

1. ホーム画面の **歯車 ⚙** で **ワーカーフォルダ**（`worker` のパス）・Python・Tesseract を設定
2. **チーム名／メンバー（RiotID）／カラー／アイコン**を設定
3. TFTを**観戦開始** → ゲーム検知でオーバーレイ表示＋ワーカー自動起動
4. 名前が読めていない場合は、ホームの **「プレイヤー対応」「位置から登録」「再OCR」** で手動補正

## ディレクトリ構成

| パス | 役割 |
|---|---|
| `manifest.json` | Overwolfアプリ定義（ウィンドウ・GEP・C#プラグイン） |
| `home.html` / `js/home.js` | ホーム画面（設定・OCRリカバリUI） |
| `in_game.html` / `js/in_game.js` | オーバーレイ（HP・生存・マーカー描画） |
| `js/background.js` | ゲーム検知・ワーカー起動/停止 |
| `plugins/worker-launcher.dll` | ワーカー起動用 C#プラグイン（ビルド済み） |
| `plugin/` | 上記C#プラグインのソース |
| `worker/` | Python OCRワーカー（`worker.py` / `ocr_engine.py` / `stats_server.py`） |

---

## 制限・注意

- **1920×1080 専用**（プレイヤーリストの座標が固定）
- TFTは**ボーダーレス／ウィンドウ表示**を想定
- OCRは画面状態に依存するため、読めない時は手動リカバリで補う設計

---

## ブランチ

- `main` … Overwolf版（本体）
- `electron-overlay` … Overwolf非依存のElectron版（実験的。ライブクライアントデータAPIで生存取得・透明ウィンドウでオーバーレイ）
